# 🤖 n8n AI Agent Mastery

This repository is dedicated to building and deploying autonomous AI Agents using n8n. From simple chatbots to complex multi-agent systems, find the blueprints and JSON templates you need here.

---

## 🌟 Key Features
*   **AI Agent Blueprints:** Templates for Research, Content Creation, and Lead Gen agents.
*   **LangChain Integration:** Examples of using Memory, Vector Stores, and Tooling.
*   **API Connectors:** Ready-made nodes for OpenAI, Anthropic, and Google Gemini.

## 📁 Popular Agent Workflows
### 1. The Research Agent
An agent that searches the web, summarizes articles, and sends a report to your Slack or Email.
*   *Nodes used:* AI Agent, Google Search Tool, Gmail.

### 2. Customer Support Bot
A 24/7 bot that reads your company’s documentation and answers user queries.
*   *Nodes used:* AI Agent, Pinecone/Supabase (Vector Store), Telegram.

### 3. Social Media Manager
Automatically generates viral content based on trending topics and schedules posts.
*   *Nodes used:* AI Agent, HTTP Request (API), Buffer/LinkedIn.

## 🚀 How to Import
1. Download the `.json` file from the `/workflows` folder.
2. Open your n8n instance.
3. Click on **Import from File** and select the downloaded JSON.
4. Add your API Credentials, and you are ready to go!
